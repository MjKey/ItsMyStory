# Хочешь помочь?

Круто, что ты здесь. Проект в активной разработке, и любая помощь — это плюс.

## Как можно помочь

**Код** — если умеешь в Java/Fabric, смотри issues или предлагай свои идеи. Перед большими изменениями лучше сначала обсудить.

**Баги** — нашёл что-то сломанное? Создай issue с описанием: что делал, что ожидал, что получил.

**Идеи** — есть мысли, как сделать лучше? Пиши в issues. Не все идеи попадут в мод, но все будут рассмотрены.

**Документация** — если что-то непонятно или можно объяснить лучше — это тоже вклад.

**Тестирование** — просто пользуйся модом и сообщай о проблемах.

## Как запустить локально

```bash
# Клонируй репозиторий
git clone https://github.com/MjKey/ItsMyStory.git
cd ItsMyStory

# Собери проект
./gradlew build

# Запусти клиент с модом
./gradlew runClient
```

Нужен Java 21.

## Pull Request

1. Форкни репозиторий
2. Создай ветку под свои изменения
3. Сделай изменения
4. Проверь, что `./gradlew build` проходит без ошибок
5. Создай PR с описанием того, что сделал и зачем

Не надо:
- Менять форматирование кода без причины
- Добавлять зависимости без обсуждения
- Делать огромные PR на 50 файлов — лучше несколько маленьких

## Стиль кода

Ничего особенного — просто пиши читаемо и следуй тому, что уже есть в проекте. Если IDE ругается — поправь.

## Вопросы?

Создай issue или напиши напрямую. Разберёмся.
