# Changelog

## [1.0.0] - 2025-11-29

### Added
- Syntax highlighting for Storykee language
  - Keywords, declarations, operators
  - Strings, numbers, comments
  - Built-in functions
  - Java section support
- IntelliSense / Autocomplete
  - Built-in functions with documentation
  - Keywords and events
  - Context-aware property suggestions
- Code snippets for common patterns
  - NPC, dialogue, quest declarations
  - Event handlers
  - Functions and control structures
- Hover information for functions and keywords
- Signature help for function parameters
- Go to Definition support
- Language configuration
  - Auto-closing brackets and quotes
  - Code folding
  - Indentation rules
