## Короче, о чём это

Устал от того, что для создания нормальной карты или модпака нужно быть программистом? Мы тоже.

It's My Story — это мод, который даёт тебе инструменты для создания своего контента: диалоги, NPC, кастомные GUI и HUD, ивенты, катсцены, квесты — и всё это через простые скрипты на `.skee`. Никакой Java, никакого ада с датапаками.

```js
npc "Торговец" {
    name: "Барыга Василий"
    
    on interact {
        dialogue("Чё надо?");
    }
}
```

Да, вот так просто. Пишешь скрипт — получаешь результат.

## Для кого это

- Создатели карт, которые хотят больше, чем командные блоки
- Разработчики модпаков, которым нужен сюжет
- Все, кто хочет оживить свой мир без боли

## Что уже есть (или почти есть)

- Диалоги с ветвлениями и выборами
- NPC, которые умеют больше, чем стоять
- Кастомные GUI и HUD
- Система ивентов
- Катсцены с управлением камерой
- Квесты
- Hot-reload — меняй скрипты без перезапуска

## Статус

**Мод в активной разработке.** Код пишется, баги правятся, фичи добавляются. Релиза пока нет, но он будет. Обещаю.
